LED connection

BCM 18: https://pinout.xyz/pinout/pin12_gpio18
- run: python led.py to test led

Button connection
using pin (GND and BCM 17)
GND: https://pinout.xyz/pinout/ground
BCM 17: https://pinout.xyz/pinout/pin11_gpio17
- run: python button.py
